# Facebook Content Pillars

## Overview

This document covers content pillars for two Facebook Pages:
1. **Facebook Tech** - Tech news, tutorials, tools for Vietnamese developers
2. **Facebook Chinese** - Chinese language learning for Vietnamese speakers (HSK 1-4)

---

## Facebook Tech Page

| Pillar | % | Posting Day | Time | Prompt Template |
|--------|---|-------------|------|-----------------|
| Tech News | 35% | Mon, Wed | 08:00 | `fb_tech_news` |
| Tutorials | 30% | Tue, Sat | 12:00 / 10:00 | `fb_tech_tutorial` |
| Tools & Tips | 20% | Thu | 12:00 | `fb_tech_tools` |
| Community | 15% | Fri | 18:00 | `fb_tech_community` |

### Posting Schedule

```
Week N:
├── Monday    08:00 → Tech News
├── Tuesday   12:00 → Tutorial
├── Wednesday 08:00 → Tech News
├── Thursday  12:00 → Tools & Tips
├── Friday    18:00 → Community
└── Saturday  10:00 → Tutorial
```

**Why these times?**
- 8AM: Developers checking feeds before work
- 12PM: Lunch break browsing
- 6PM: Evening wind-down, more engagement with discussion posts
- Saturday 10AM: Weekend learners catching up

### Pillar 1: Tech News (35%)

**Focus:** AI updates, new releases, industry trends, breaking tech news

**Topic Examples:**
- AI coding assistants comparison (Copilot vs Claude Code)
- New framework releases and what they mean
- Cloud vs local LLM developments
- Security news affecting developers

**Tone:** Casual, opinionated, conversational
**Hook Types:** Hot take, "Vừa thấy tin này...", surprising stat

**Post Structure:**
```
[Emoji + Hook: tin nóng hoặc câu hỏi]

[Tóm tắt 2-3 điểm chính]

[Ý kiến cá nhân]

[Câu hỏi thảo luận]

#hashtags
```

### Pillar 2: Tutorials (30%)

**Focus:** Step-by-step guides, code examples, practical how-tos

**Topic Examples:**
- Docker basics for beginners
- Git workflow for small teams
- Setting up development environment
- API integration tutorials

**Tone:** Friendly teacher, "mình chỉ bạn nhé"
**Hook Types:** "Bạn đã bao giờ...", problem statement, "Chỉ 5 phút..."

**Post Structure:**
```
[Hook: vấn đề thường gặp]

[Bước 1-3/5 giải quyết]
[Code example nếu có]

[Tips bonus]

"Save post này để tham khảo sau nhé!"
#hashtags
```

### Pillar 3: Tools & Tips (20%)

**Focus:** Tool reviews, productivity tips, developer utilities

**Topic Examples:**
- VS Code extensions must-have
- Free hosting platforms comparison
- CLI tools that save time
- Browser dev tools tricks

**Tone:** Honest reviewer, sharing personal experience
**Hook Types:** "Mình vừa thử tool này...", before/after comparison

### Pillar 4: Community (15%)

**Focus:** Polls, discussions, Q&A, community engagement

**Topic Examples:**
- IDE preference polls
- Framework debates
- Career path discussions
- "Bạn đang học gì?" threads

**Tone:** Curious, inclusive, encouraging participation
**Hook Types:** Direct question, "Mọi người nghĩ sao về...", poll format

### Posting Schedule

```

Weekly additions:
├── Monday    10:00 → Grammar
├── Tuesday   18:00 → Practice/Quiz
├── Wednesday 10:00 → Grammar
├── Friday    18:00 → Practice/Quiz
└── Saturday  10:00 → Culture
```

**Why these times?**
- 7AM: Morning learning habit - start the day with a new word
- 10AM: Mid-morning focus time for grammar study
- 6PM: Evening engagement for interactive quizzes
- Saturday 10AM: Weekend cultural exploration

### Pillar 1: Vocabulary (35%)

**Focus:** Daily word learning, HSK 1-4 vocabulary with pinyin and examples

**Topic Examples:**
- Basic greetings and expressions
- Numbers and counting
- Shopping and money vocabulary
- Technology words in Chinese

**Tone:** Encouraging teacher, "Hôm nay mình học từ mới nhé!"
**Format:** Hán tự + Pinyin + Vietnamese meaning + example sentences

**Post Structure:**
```
[Emoji] Từ mới hôm nay!

[Hán tự] ([Pinyin]) = [Nghĩa tiếng Việt]

Ví dụ:
1. [Câu ví dụ + pinyin + dịch]
2. [Câu ví dụ + pinyin + dịch]

Mẹo nhớ: [memory tip]

Từ liên quan: [related words]

#HọcTiếngTrung #HSK #TừVựng
```

### Pillar 2: Grammar (25%)

**Focus:** Chinese grammar patterns, comparison with Vietnamese, common mistakes

**Topic Examples:**
- 是...的 structure for emphasis
- 了 particle usage
- Measure words (量词)
- Sentence order differences

**Tone:** Patient explainer, relatable comparisons to Vietnamese
**Format:** Pattern + examples + Vietnamese comparison + mini exercise

### Pillar 3: Culture (20%)

**Focus:** Chinese culture topics paired with relevant vocabulary

**Topic Examples:**
- Mid-Autumn Festival traditions
- Chinese tea culture
- Chinese New Year customs
- Food culture and vocabulary

**Tone:** Storyteller, fascinating facts, cultural bridge
**Format:** Cultural story + 3-5 related vocabulary words

### Pillar 4: Practice (20%)

**Focus:** Interactive quizzes, fill-in-the-blank, multiple choice

**Topic Examples:**
- Fruit vocabulary quiz
- Complete the sentence exercises
- Pinyin matching games
- Translation challenges

**Tone:** Fun, gamified, encouraging participation
**Format:** 3-5 questions + answer in first comment

## Content Quality Checklist

### Facebook Tech - Must Have
- [ ] Compelling hook (first 2 lines)
- [ ] Clear value for the reader
- [ ] Appropriate length (100-200 words)
- [ ] 3-5 relevant hashtags
- [ ] Call-to-action or discussion question
- [ ] 2-4 emoji (not excessive)
- [ ] Vietnamese with English tech terms

### Facebook Tech - Must NOT Have
- [ ] "As an AI..." or AI artifacts
- [ ] Overly formal/LinkedIn tone
- [ ] More than 5 hashtags
- [ ] Walls of text without formatting
- [ ] Clickbait without substance

### Facebook Chinese - Must Have
- [ ] Pinyin for ALL Chinese characters
- [ ] Vietnamese translation for all examples
- [ ] Correct tone marks in pinyin
- [ ] HSK level appropriate content
- [ ] Encouraging, supportive tone
- [ ] 150-250 words
- [ ] Memory tips or mnemonics

### Facebook Chinese - Must NOT Have
- [ ] Chinese without pinyin
- [ ] Incorrect pinyin/tones
- [ ] Content above HSK 4 level without explanation
- [ ] AI-sounding language
- [ ] Grammar errors in Vietnamese

## Prompt Optimization Guide

### Facebook Tech

**If output is too formal (LinkedIn-like):**
1. Add "Viết như đang chat với bạn bè developer" to context
2. Reduce word count to 100-150
3. Add more emoji instructions

**If output lacks engagement:**
1. Emphasize the discussion question at the end
2. Add "Đưa ra 2-3 options cho reader chọn"
3. Use more direct questions

### Facebook Chinese

**If Chinese content quality is poor:**
1. Consider switching to Qwen2 7B for Chinese generation
2. Simplify to HSK 1-2 level only
3. Add more pinyin validation in the prompt
4. Include reference examples in the context field

**If content is too textbook-like:**
1. Add "Viết như giáo viên đang chat với học sinh"
2. Include real-life scenarios
3. Add fun memory tricks and mnemonics
