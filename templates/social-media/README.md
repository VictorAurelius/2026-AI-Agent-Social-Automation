## Brand Guidelines

### Colors
- Primary: `#2563EB` (Blue)
- Secondary: `#10B981` (Green)
- Accent: `#F59E0B` (Amber)
- Dark BG: `#1E293B`
- Light BG: `#F8FAFC`

### Fonts
- Title: Noto Sans Bold
- Body: Noto Sans Regular
- Code: JetBrains Mono
