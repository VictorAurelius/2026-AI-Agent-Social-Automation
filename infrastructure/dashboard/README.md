# AI Agent Dashboard

Lightweight web dashboard for content management.

## Tech Stack
- HTML5 + Tailwind CSS (CDN)
- Alpine.js (CDN, no build)
- n8n webhooks as API backend

## Setup
1. Open `index.html` in browser (or serve via nginx)
2. Configure API endpoint in `index.html` (n8n webhook URL)
3. Create n8n webhook workflows for API endpoints

## Status: Design Phase
Currently a static prototype. Requires n8n webhook API implementation.
